## this

`this` 는 자신이 속한 객체 또는 자신이 생성할 인스턴스를 가리키는 자기참조변수다.
단 `this` 는 함수 호출 방식에 의해 동적으로 결정된다.

1. 일반 함수(중첩 함수, 콜백 함수 포함)로 호출된 모든 함수의 내부 this 는 전역 객체가 바인딩된다. 클로저를 이용하거나, call, apply, bind 를 통해 호출해서 this 를 제어할 수 있다.

2. 메서드 내부의 this 는 프로퍼티로 메서드를 가리키고 있는 객체와는 관계없이 메서드를 호출한 객체에 바인딩된다.

```
const person = { name: ‘lee’, getName() { return this.name } }
Person.getName() // lee
```

3. 생성자 함수 내부의 this 에는 생성자 함수가 생성할 인스턴스가 바인딩된다.
